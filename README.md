# 🎮 Therapy Funnel: Создай своего героя

Я сделал этот тест, потому что мне всегда было интересно: почему одним людям помогает один подход к терапии, а другим — совсем другой. Вместо скучной анкеты получилась игра: вы создаёте персонажа, а по его чертам становится видно, какая школа психотерапии вам ближе.

## Зачем это нужно?

Никакой науки — просто интересно. Десять вопросов, четыре варианта ответа. Отвечайте то, что первым приходит в голову. Не пытайтесь угадать «правильный» ответ — его нет.

## Что внутри

- **10 вопросов** в формате «Создай своего героя» — никаких скучных анкет.
- **Ответы в случайном порядке** — вы не догадаетесь, какая школа к чему относится.
- **Четыре школы терапии** — КПТ, психоанализ, гештальт и экзистенциальная терапия.
- **Результат с характером** — вы получите не просто сухие баллы, а описание героя и пояснение, почему вам может подойти именно этот подход.
- **Работает на всём** — от телефона до компьютера, без сервера и регистрации.

## Как это работает

### Вопросы
Десять вопросов, четыре варианта ответа в каждом. Варианты перемешаны так, что вы не догадаетесь, какой ответ к какой школе относится. 

### Подсчёт
Каждый ответ добавляет один балл школе терапии, которая за ним скрывается. Школы обозначены буквами:
- `К` — Когнитивно-поведенческая терапия (КПТ)
- `П` — Психоанализ  
- `Г` — Гештальт-терапия
- `Э` — Экзистенциальная терапия

### Результат
После последнего вопроса вы увидите:
1. **Имя и описание вашего героя** — один из четырёх архетипов.
2. **Сколько баллов набрала каждая школа** — наглядно, с цветами.
3. **Рекомендацию** — почему вам может подойти именно этот подход.
4. **Что с этим делать** — простые советы, как использовать результат.

## Четыре архетипа

В зависимости от того, какие ответы вы чаще выбирали, вы получите одного из четырёх персонажей:

| Школа | Архетип | Что это значит |
|-------|---------|----------------|
| **КПТ** | **СТРАТЕГ** | Вы мыслите ясно и пошагово. Для вас проблемы — это задачи, у которых есть решение, и вы знаете, как к нему подступиться. |
| **Психоанализ** | **АРХЕОЛОГ ТЕНЕЙ** | Вы ищете глубину и причину. Вам интересно, откуда что берётся, и вы готовы копать в прошлое, чтобы понять настоящее. |
| **Гештальт** | **ЧУВСТВУЮЩИЙ КЛИНОК** | Вы доверяете телу и моменту. Вам важно, что происходит прямо сейчас, и вы умеете слушать свои ощущения. |
| **Экзистенциальная** | **СТРАННИК БЕЗ КАРТЫ** | Вы задаёте большие вопросы. Смысл, свобода, одиночество — для вас это не абстракции, а то, с чем приходится иметь дело каждый день. |

## Если захотите что-то поменять

### Вопросы
Откройте `index.html`, найдите массив `questions` и отредактируйте:

```javascript
const questions = [
  {
    text: "Ваш вопрос здесь",
    options: [
      { text: "Вариант А", school: "К" },
      { text: "Вариант Б", school: "П" },
      { text: "Вариант В", school: "Г" },
      { text: "Вариант Г", school: "Э" }
    ]
  }
  // ... остальные вопросы
];
```

### Внешний вид
Все стили — в теге `<style>` того же `index.html`. Основные цвета и размеры заданы CSS-переменными:

```css
:root {
  --primary: #4a6fa5;     /* Основной цвет */
  --secondary: #166088;   /* Второстепенный цвет */
  --accent: #ff6b6b;      /* Цвет акцентов */
  --light: #f8f9fa;       /* Светлый фон */
  --dark: #343a40;        /* Тёмный текст */
  --shadow: 0 4px 12px rgba(0, 0, 0, 0.1); /* Тени */
  --radius: 12px;         /* Скругления углов */
}
```

### Аналитика
Если захотите отслеживать, сколько людей прошло тест, добавьте перед `</body>` в `index.html`:

```html
<script>
  // Google Analytics
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'ВАШ-ID-OT-GOOGLE');
</script>
```
