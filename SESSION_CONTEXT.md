## 💬 История решений

### Архитектура:
- **Frontend:** HTML5/CSS3/JS (vanilla, без фреймворков)
- **Backend:** Node.js + Express (server.js)
- **Database:** SQLite → PostgreSQL
- **Hosting:** GitHub Pages (frontend) + Render/Heroku (backend)
- **Auth:** JWT + localStorage

### Бизнес-модель:
- **Фаза 1:** Бесплатный скрининг (10 вопросов, 5 школ)
- **Фаза 2:** Расширенный тест (20 вопросов, только для авторизованных)
- **Фаза 3:** Платная подписка $9.99/мес + AI-терапевт

### Дизайн:
- **Символ:** Сигил Люцифера (найден на zcache, скачан локально)
- **Цвета:** Тёмно-красный (#8b0000), акцент оранжевый (#ff4500)
- **Тема:** Сакральная геометрия, готический стиль
- **Анимации:** Hover-эффекты, пульсация свечения

### Вопросы (критика и исправления):
- **Проблема:** вопросы были «топорными», ответы явно выдавали школы
- **Решение:** полная переработка 10 вопросов
- **Фидбэк от психолога Андрея Труфанова:** учтён
- **Новый подход:** ответы как поведенческие стили, соответствие школ скрыто

## ⚠️ Текущие проблемы и риски

1. **Изображение сигила** — загружено с zcache, может быть нарушение авторских прав
2. **SEO** — title использует символ ✸, который может не читаться поисковиками
3. **Мобильная версия** — тестирована не на всех устройствах
4. **Юридические аспекты** — отсутствуют политика конфиденциальности и условия использования
5. **Платежи** — не настроены, нет интеграции с Stripe/ЮKassa
